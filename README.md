# SpongeBob Endless Runner

![Java](https://img.shields.io)
![NetBeans](https://img.shields.io)

A completed solo school project featuring a SpongeBob-themed endless runner. This game was built from the ground up to demonstrate core Object-Oriented Programming (OOP) principles, including inheritance, encapsulation, and modular state management.

## Game Features
* **Dynamic Obstacle Spawning** Utilizes a randomized spawning system to keep gameplay challenging
* **Modular Entity System** Character and obstacle classes built on a scalable hierarchy
* **Collision Detection** Custom-built logic handling interaction between SpongeBob and obstacles
* **Game State Management** Includes distinct logic for menu, active play, and game over screens

## Technical Implementation
* **Language** Java
* **IDE** Developed and tested in Apache NetBeans
* **Architecture** Focused on OOP design patterns for clean and maintainable code
